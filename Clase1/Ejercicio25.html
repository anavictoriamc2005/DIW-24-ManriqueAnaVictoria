• HTML: Lenguaje de marcado utilizado para estructurar contenido en la web.
  • CSS: Lenguaje usado para describir la presentación de un documento escrito en HTML o XML.
  • JavaScript: Lenguaje de programación utilizado para crear contenido interactivo en la web.
  • SEO: Conjunto de estrategias y prácticas utilizadas para mejorar la visibilidad de una página web en los motores de búsqueda.
  • API: Conjunto de definiciones y protocolos que permiten la comunicación entre diferentes aplicaciones de software.
